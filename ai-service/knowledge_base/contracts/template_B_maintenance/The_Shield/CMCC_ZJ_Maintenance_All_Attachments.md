## 附件1：中国移动浙江公司2023年至2026年网络综合代维集中采购技术规范书

### 1. 总则
为确保中国移动通信集团浙江有限公司（以下简称“甲方”）网络综合代维工作的规范化、标准化，明确代维服务的技术要求，特制定本技术规范书。乙方（代维单位）必须严格遵守本规范书的要求，提供高质量的代维服务。

### 2. 服务界面划分
*   **光缆线路维护**：以 ODF 架外侧端口为界，ODF 架以内（含法兰盘）由设备专业维护，ODF 架以外（含尾纤、连接器）至光缆线路侧由线路专业维护。
*   **基站设备维护**：以基站机房（或一体化机柜）市电引入配电箱（电表箱）下口为界，电表箱及上游线路归局方/铁塔公司负责，电表箱下口至基站主设备（含开关电源、蓄电池、空调等）由代维单位负责。

### 3. 巡检周期与标准
乙方需按照以下频次对甲方资产进行巡回检查，确保设备设施处于良好运行状态：

| 资产类别 | 巡检项目 | 巡检周期 | 作业标准 (SOP) |
| :--- | :--- | :--- | :--- |
| **汇聚机房** | 环境/动力/主设备 | **每周 1 次** | 检查机房温度(20-25℃)、湿度、清洁度；测试蓄电池电压；检查灭火器有效期。 |
| **普通基站** | 铁塔/天馈/机柜 | **每月 1 次** | 检查塔体垂直度、螺栓紧固；天线方位角/俯仰角核对；机柜除尘；防盗门锁检查。 |
| **杆路/管道** | 路由/人井/挂钩 | **每季度 1 次** | 路由全覆盖巡检；检查线缆垂度；人井抽水清淤；挂钩补缺；警示牌补漆。 |

### 4. 故障修复时限 (SLA)
乙方必须建立 7*24 小时响应机制，严格执行以下故障修复时限要求：

| 故障等级 | 定义描述 | 响应时限 | **修复时限 (SLA)** |
| :--- | :--- | :--- | :--- |
| **一级故障** | 造成大面积断网（汇聚环中断、汇聚机房掉站等） | 15分钟 | **≤ 2小时** |
| **二级故障** | 单个基站退服、单条重要专线中断 | 30分钟 | **≤ 4小时** |
| **家宽投诉** | 家庭宽带用户报障 | **30分钟内**联系 | **24小时内**上门修复 |

### 5. 人员资质要求
*   **特种作业证**：从事高处作业人员必须持有有效的《高处作业证》；从事电工作业人员必须持有《电工进网作业许可证》。
*   **光缆熔接**：光缆维护人员必须持有光缆熔接技能认证证书。
*   **持证上岗率**：项目执行期间，乙方所有技术岗位人员持证上岗率必须达到 **100%**。

## 附件2：中国移动通信集团浙江有限公司网络代维考核细则

### 1. 考核原则
考核采用“月度打分制”，满分 100 分。考核结果直接与当月代维费用结算挂钩。

### 2. 付款挂钩规则
| 多维度综合评分 (S) | 结算比例 | 备注 |
| :--- | :--- | :--- |
| **S ≥ 95** | 100% | 全额支付 |
| **85 ≤ S < 95** | 90% | 扣除 10% 服务费 |
| **70 ≤ S < 85** | 70% | 扣除 30% 服务费，并下发整改通知书 |
| **S < 70** | **熔断/终止** | **不予支付当月服务费，甲方有权单方面终止合同** |

### 3. 关键 KPI 指标 (Key Performance Indicators)
| 指标名称 | 目标值 | 扣分标准 (Penalty Ratios) |
| :--- | :--- | :--- |
| **基站可用率** | ≥ 99.9% | 每低于 0.1%，扣 **2分**。 |
| **发电及时率** | ≥ 98% | 低于目标值，每发生一次超时，扣 **500元**（直接扣款）。 |
| **工单处理及时率** | ≥ 99% | 每超时一张工单，扣 **100元**。 |
| **巡检完成率** | 100% | 未完成按比例扣分；**造假行为（如PS照片）**：**扣除当月代维费总额的 10%**，并全省通报。 |

## 附件4：中国移动通信集团浙江有限公司网络代维管理办法

### 1. 组织架构要求
*   乙方必须在甲方各分公司所在地市设立办事处或分公司，配备专职项目经理及相应的管理团队。
*   项目经理必须具备 PMP 或同等资质，且具有 3 年以上通信代维管理经验。

### 2. 车辆与装备配置
*   **抢修车辆**：每 30 个物理基站或 每 500 公里光缆，必须配置 1 辆专用抢修车。车辆必须统一涂刷“中国移动通信代维”标识。
*   **工器具**：每组抢修人员必须配备熔接机、OTDR、光功率计、红光笔、发电机（3KW/5KW）等全套仪表工具。

### 3. 甲供材管理（严禁流失）
*   甲方提供的光缆、尾纤、光猫、机顶盒等物资（甲供材），乙方必须建立严格的《进销存台账》。
*   **盘亏处理**：每季度进行一次盘点，如发现甲供材短少或非正常损耗，乙方须按**采购原价的 2 倍**进行赔偿。

## 附件5：中国移动网络代维质量规范

### 1. 机房环境标准
*   **温度/湿度**：温度保持在 20-25℃，相对湿度 40%-70%。
*   **清洁度**：地面无灰尘，机柜无积灰，无鼠迹、无虫网。
*   **封堵**：所有进出线孔洞必须使用防火泥严密封堵，防止小动物进入。

### 2. 线缆工艺标准
*   **标签规范**：所有线缆（电源线、光纤、网线）两端必须粘贴防水、防撕裂的专用标签，标签内容（起止点、用途）必须清晰准确。
*   **绑扎规范**：线缆布放应横平竖直，转弯处曲率半径符合规范（光纤 > 40mm），扎带间距均匀，剪断处平整无尖角。
*   **光缆接头盒**：接头盒内光纤盘绕整齐，无挤压；加强芯必须固定牢固；接头盒密封良好，防水测试合格。

## 附件6：中国移动网络代维服务管控指南

### 1. 电子运维系统执行要求
*   **工单流转**：乙方必须通过甲方的“电子运维系统”（EOMS/IOSM）接收和回单。**严禁**线下口头派单、电话派单，无系统工单的作业将不予结算费用。
*   **流程规范**：接单 -> 预约 -> 到位 -> 处理 -> 验证 -> 归档。每个环节必须在系统中实时操作。

### 2. 也是 GPS 打卡与轨迹管理
*   **到位打卡**：代维人员到达故障现场或巡检点后，必须通过手持终端 APP 进行 GPS 打卡签到。
*   **误差范围**：打卡坐标与资源点实际坐标距离偏差不得超过 **500米**。
*   **轨迹核查**：甲方将不定期抽查人员移动轨迹，如发现“人机分离”或“虚假打卡”，按数据造假条款处理。

### 3. 夜间作业管理
*   凡在 **22:00 至 次日 06:00** 期间进入核心机房或进行可能影响业务的操作，乙方必须提前向甲方提交《夜间作业申请单》，获批后并办理《夜间通行证》方可实施。

## 附件8：数据安全和保密协议

### 1. 保密信息范围
本协议所称保密信息包括但不限于：
*   **网络数据**：网络拓扑图、IP地址规划、路由配置、系统账号密码。
*   **用户数据**：客户姓名、身份证号、手机号、家庭住址、宽带账号、通话记录、上网日志。
*   **经营数据**：资费套餐、营销策划、未公开的财务数据。

### 2. 数据主权与存储
*   所有甲方数据的所有权归中国移动所有。
*   **本地化存储**：乙方在处理甲方数据时，严禁将其存储在乙方私有云、第三方公有云或 **境外服务器** 上。
*   严禁将甲方数据带出甲方指定的安全作业区域。

### 3. 数据销毁
合同终止或项目结束后，乙方必须在 **5个工作日内** 销毁所有载体上的保密信息，确保不可恢复，并向甲方出具《数据销毁书面承诺书》。

### 4. 违约责任
若发生数据泄露事件，乙方应承担由此产生的一切法律后果和经济损失（包括但不限于对客户的赔偿、行政罚款、商誉损失），并支付不低于 **50万元/次** 的违约金。

## 附件9：数据委托处理协议

根据《中华人民共和国个人信息保护法》（PIPL）及相关法规，双方达成如下协议：

### 1. 委托处理目的与范围
乙方仅在履行“家宽安装、故障维修”等合同义务所必需的范围内，处理甲方客户个人信息。**严禁**超出此范围收集、使用客户信息。

### 2. 最小化原则
乙方在服务过程中，不得收集与维修无关的生物识别信息（如人脸、指纹、声纹）或敏感个人信息。

### 3. 安全义务
乙方应建立完善的个人信息保护制度，对接触客户信息的员工进行背景调查和保密培训，并签署个人保密承诺书。

### 4. 违约处罚
如乙方或其员工利用职务之便，非法出售、提供或公开甲方客户个人信息：
1.  乙方承担全部民事、行政及刑事责任。
2.  乙方负责全额赔偿甲方及受害客户的所有损失。
3.  甲方有权立即解除合同并扣除所有未结费用。

## 附件10：安全生产责任书

### 1. 安全主体责任
乙方是本项目代维作业现场安全生产的**第一责任人**，对代维人员的人身安全及作业过程中的第三方安全负全面责任。

### 2. 全责条款
凡在代维服务过程中发生的人身伤亡、交通事故、火灾、触电、高坠等安全事故，**均由乙方自行处理并承担全部法律责任和经济赔偿责任**。甲方不承担任何连带责任。如甲方因此被判承担责任的，甲方有权向乙方全额追偿。

### 3. 保险强制要求
乙方必须为全体代维人员购买足额的商业保险。
*   **意外伤害险**：保额不低于 **80万元/人**。

## 附件14：中国移动网络代维考核规范 (通用框架)

### 1. 考核维度
集团层面统一建立代维绩效评估体系，包含四个维度：
*   **资源投入 (20%)**：人员、车辆、仪表到位率。
*   **维护质量 (40%)**：故障修复时长、网络可用率、巡检合格率。
*   **服务感知 (20%)**：客户满意度、投诉率。
*   **综合管理 (20%)**：安全生产、报表提交、配合度。

### 2. 信用评价体系
根据年度考核结果，对代维单位进行信用分级：
*   **A级**：优秀，优先续签，增加市场份额。
*   **B级**：良好，维持现有份额。
*   **C级**：合格，缩减市场份额，黄牌警告。
*   **D级**：不合格，**终止合同，列入黑名单**。
